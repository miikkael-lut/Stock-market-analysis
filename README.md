# Stock-market-analysis


This analysis is done with Google Colab notebook using python. I have created this analysis which takes in stock values from 2016 downloaded from Yahoo finance and produces report with displays key indicators of the stock market movements for chosen stoc. Then uses 5 different machine learning models to make predictions for stock performance. Model uses machine learning models ANN (Artificlal eural etwork), LSTM (Long short-term memory), Random Forrest, Decicion tree and Linear regression.
